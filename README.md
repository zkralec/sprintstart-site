# Sprint Start Site

This is a minimal GitHub Pages site for Sprint Start.

## Files
- `privacy.html`
- `support.html`

## Publish on GitHub Pages
1. Create a new GitHub repository, for example `sprintstart-site`
2. Upload these files to the repository root
3. Go to **Settings > Pages**
4. Under **Build and deployment**, choose:
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
5. Save
6. Your pages will publish at a URL like:
   - `https://YOUR-USERNAME.github.io/sprintstart-site/privacy.html`
   - `https://YOUR-USERNAME.github.io/sprintstart-site/support.html`

## Replace before publishing
Search for `YOUR-EMAIL-HERE` and replace it with your real support email address.
